# StructureWall

<img src="img/wall.png" alt="" align="right" />

阻挡所有creep的移动。
玩家可以在已控制的房间中建造可破坏的构筑墙。
有些房间还包含不可破坏的构筑墙，将新手区或重生区与世界其他地方隔开，又或将新手/重生区划分为较小的区域。不可破坏的构筑墙没有`hits`属性。
